◆ buildTail()

static UnsolicitedResponseTail buildTail ( UnsolicitedResponseType  urt)
static

Builds a frame tail.

Since the preamble of an unsollicited response is always identical (except for the ALP cmd length byte) one can simply add a different tail to the preamble. In order to do that several types of frames where defined.

Parameters
urtEnumeration indicating the frame (tail) type.
Returns
Returns a frame tail structure containing data and length.
Note
The data pointer needs to be freed after transmission.

Transmit an unsolicited response over the D7 interface.

In order to transmit a message over the D7 RF link a serial frame is built containing all necessary information for the receiving D7 modem in order to propagate (a part of) said frame over the D7 RF link.

Parameters
dataPointer to data buffer, will be freed.
dataLengthAmount of bytes in data buffer.
Returns
Returns 1 if successful and 0 otherwise.
